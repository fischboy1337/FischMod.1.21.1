# FischMod 1.21.1 (DISCONTINUED)
FischMod is an discontinued PVP client for [1.8.9]() or 1.21.1 made for friends.

## Are there cosmetics?
No.

## What features does the Client has?
- Armor Status
- Coords Overlay
- FPS Overlay
- Keystrokes
- ToggleSprint

## How to I open the ClickGUI?
press R-Shift
